## demo说明

* demo1: 简单的字符串解析
* demo2: 利用phamtoJs库执行JS，用cheerio选中目标
* demo3: 加async控制请求数
* demo4: 查找加载数据的url请求，直接得到json格式数据
* demo5: 大量数据用moongodb保存
* demo6: 使用rsa加密密码，模拟百度登录
* demo7: 同demo3
